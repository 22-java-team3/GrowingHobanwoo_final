# <b>호반우를 키워라</b>

## <b>프로젝트 개요</b>

경북대학교 2022년 1학기 수업 자바프로그래밍(COMP0217-005)의 Term Project로 시작한 시뮬레이션 게임 <b>호반우를 키워라</b>의 README 파일입니다.<br>
경북대학교 공식 마스코트인 ‘호반우’를 캐릭터 이미지로 사용하여 학교 홍보를 기대할 수 있습니다.

## <b>프로젝트 구성</b>

플레이어는 경북대학교 학생의 삶을 체험하면서 자신의 선택으로 캐릭터의 엔딩을 바꿀 수 있습니다.<br>
어떤 활동을 하느냐에 따라 <u>체력,</u> <u>스트레스,</u> <u>행복도</u>가 변화하며 이 수치들은 엔딩에 영향을 미칩니다.

### <b>학습</b>

게임을 처음 시작했을 때 수강신청을 통해 한 학기 동안 공부할 과목을 선택할 수 있습니다.<br>
<u>강의실</u>에서 수업에 출석하고, 필기 노트를 작성하며 공부합니다.<br>
<u>도서관</u>에서 노트를 고쳐쓰면서 복습하고, 단어 입력 게임을 통해 내용을 머리에 새길 수 있습니다.<br>
실제 경북대학교 컴퓨터학부 글로벌SW융합전공 개설 과목들을 게임에 반영하여 몰입감을 높이고, 동시에 전공 학습까지 가능합니다.

### <b>자기계발</b>

<u>체육관</u>에서 운동을 통해 건강을 관리합시다.<br>
<u>동아리</u>에서 추가적인 공부를 하거나, 그림을 그리며 여가를 즐깁시다.

### <b>생활</b>

학교생활을 하기 위해 꼭 필요한 체력은 <u>식당</u>에서 음식을 먹어 보충할 수 있습니다.<br>
게임에서 섭취한 음식과 비례하여 체력이 회복됩니다.

## <b>프로젝트 환경 및 사용법</b>

Java SE 8 이상<br>
Eclipse를 추천합니다.<br>
한글이 깨질 경우, 해당 클래스의 text file encoding을 UTF-8 혹은 MS949로 변경해주세요.

## <b>주의사항</b>

점프게임 종료시 score창이 뜬 후 닫기 버튼을 눌러 창을 닫아주세요.<br>
게임이 시작되면 끝나기 전까지 끄거나 다른 곳을 누르지 마세요.<br>
캐릭터의 변화를 보시려면 캐릭터를 눌러 상태를 확인하세요.<br>
각 활동마다 체력, 스트레스, 행복도의 영향이 다르니 주의하세요.<br>
깜짝 엔딩이 숨겨져 있습니다.<br>
엔딩이 뜨면 게임 끝내기를 눌러 게임을 끝내세요.<br>
